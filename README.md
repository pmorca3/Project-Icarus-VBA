# Project Icarus - VBA Cloud Flight Log

Sistema avanzado de gestión de combustible y tracking en tiempo real para operaciones de vuelo, conectado con **Supabase (PostgreSQL)**.

### Características
* **Cloud Sync:** Sincronización bidireccional mediante REST API.
* **Security:** Validación de operarios mediante PIN y seguridad RLS.
* **Real-time Tracker:** Integración con iOS Shortcuts para tracking GPS de la flota.
* **Auto-Auditoría:** Filtrado inteligente de logs de los últimos 2 días.

### Tecnología
* Excel VBA + JSON Converter.
* Supabase Cloud Backend.
* ServerXMLHTTP para comunicaciones asíncronas.
